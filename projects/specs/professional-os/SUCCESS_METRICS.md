# Success Metrics

## North Star Metrics

### Template (One-Time)
**Paying customers who generate at least one tailored output (resume, post, assessment) within 30 days of purchase.**

### Dashboard (Subscription)
**Monthly Recurring Revenue (MRR) from active subscribers who engage with the dashboard weekly.**

These measure both purchase/subscription AND actual usage—the true indicator of value delivered.

## How We'll Track

### Automated
- **Lemon Squeezy dashboard** - Template sales, subscription MRR, refunds, churn
- **Supabase** - User signups, tier distribution, feature usage
- **Vercel Analytics** - Dashboard traffic, session duration
- **Google Analytics** - Landing page traffic, conversions
- **GitHub** - Forks, stars as social proof

### Manual
- Customer feedback surveys (monthly)
- Support ticket analysis
- Social mentions tracking
- Job placement success tracking

### Tools
- **Sales/Subscriptions:** Lemon Squeezy
- **Auth/Database:** Supabase
- **Analytics:** Plausible or Fathom (privacy-friendly)
- **Email:** ConvertKit or Buttondown
- **Feedback:** Tally or Typeform

## Red Flags (When to Pivot)

If we see these, reconsider the approach:

| Signal | Threshold | Action |
|--------|-----------|--------|
| Month 1 template sales | <15 | Revisit positioning/pricing |
| Template refund rate | >20% | Investigate product issues |
| Dashboard free signups | <20/month | Revisit marketing |
| Free → Pro conversion | <5% | Improve upgrade prompts |
| Monthly churn | >15% | Major product issues |
| Usage rate | <20% | Improve onboarding |
| NPS | <20 | Major product changes needed |
| No referrals | 0 in 3 months | Not solving real problem |
| Auto-apply success rate | <1% | Revisit algorithm |

## Success Definition

### Minimum Success (Validated)
- 50+ template customers
- 20+ dashboard subscribers
- <10% refund rate, <10% churn
- 3+ unsolicited testimonials
- Clear path to $2k MRR

### Target Success (Sustainable)
- 100+ template customers
- 50+ dashboard subscribers
- $1,000+ MRR
- Growing 15% MRR month-over-month
- Strong testimonials for marketing
- 1+ job placement success story

### Stretch Success (Breakout)
- 250+ template customers in year 1
- 150+ dashboard subscribers
- $2,000+ MRR
- Featured in major dev publication
- Competitors start copying approach
- 5+ documented job placements via auto-apply
