## Application Strategy

### Sources to Monitor
- LinkedIn (set job alerts)
- Wellfound (AngelList) for startups
- Hacker News "Who's Hiring" monthly threads
- Remote-specific boards (WeWorkRemotely, RemoteOK)
- Direct company career pages for target companies

### Application Approach
1. Prioritize roles where job description matches 70%+ of skills
2. Tailor resume for each application using `/generate-resume` command
3. Track all applications in `applications.json`
4. Follow up after 1 week if no response
5. Prep for interviews using `interviews.json`
